Release notes for v1.2
New features
  • Added MIDI export for sequences (beta).
    Allows you to export your Blackbox patterns as standard MIDI files, so you can edit, arrange, or process them in any DAW or external sequencer.
  • Added SFZ export for pads (beta).
    Lets you export your pad settings as an SFZ instrument, making it possible to load your sounds and mappings into compatible samplers (e.g., SFZ players).
Bug fixes
  • Fixed an issue where non-integer pitch values were not previewed correctly.
  • Fixed sample loading for legacy preset format.
