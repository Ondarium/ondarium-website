Release notes for v1.5
New features
  • Expanded Audio Editor effects
    The Audio Editor now includes additional creative FX: Bit Crush, Distort, Tape Stop, Flanger, Chorus, and Phaser.
  • Pads Mode for Keyboard
    A new Pads mode allows pads to be triggered directly via MIDI notes, making it easier to play, test, and audition pad layouts from the virtual keyboard or an external MIDI controller.
  • Extended Procedural Generator sound types
    The Procedural Generator now includes new sound categories: Clap, Tom, Rim, and FX.
Improvements
  • New pad parameters: per-pad MIDI In channel and Pad Note.
  • Persistent FX parameter state: the Audio Editor now remembers the last-used parameters for each audio process.
  • Improved waveform editing precision.
Bug fixes
  • Fixed a velocity range edge case when loading presets with multi-sample pads.
