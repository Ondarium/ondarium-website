Release notes for v1.7
New features
  • Sequencer
    • Live MIDI recording
    • MIDI import: MIDI files can now be imported into the sequencer. You can select which track to import and apply options to simplify drum kit imports.
    • Note preview while editing
    • Double-click the step ruler to start playback from a specific position
    • Playhead showing the current playback position
  • New sound generators: VA Subtractive, FM (2 operators), SuperPad, Wavetable
  • New Audio Editor effects: Vocoder, Stutter
  • Slice Loop
  • Clip Mode with Time-Stretch
  • Bulk loop point import: loop points can now be automatically imported when loading WAV files.
  • Velocity modulation: pad gain can now be modulated by velocity.
  • Velocity Slider: a dedicated Velocity Slider next to the Virtual Keyboard lets you adjust the trigger velocity used when previewing pads or playing from the keyboard.
Improvements
  • Optimized sound engine (reduced CPU usage).
  • Improved choke handling.
  • Multi-sample filename parsing now supports the 3-number format: RootNote_VelocityLow_VelocityHigh
    Example: WoodenClick_OB_79_90_127.wav
  • Improved WAV metadata extraction (root note).
Changes
  • In the sequence editor, Left / Right arrow keys now select notes.
  • Use Ctrl + Left / Right to nudge notes horizontally.
Bug fixes
  • Fixed sample loading issues when multiple multi-samples are used across several pads.
  • Fixed preview voice not stopping correctly.
  • Fixed undo history being cleared when changing patterns.
