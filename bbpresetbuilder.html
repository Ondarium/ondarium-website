Overview

Blackbox Preset Editor is a dedicated tool for the 1010music Blackbox that turns preset creation into a clear, visual, and fast workflow. No more hand-editing preset.xml files: import your samples, fine-tune parameters, audition changes instantly, and save clean presets ready for your projects.

Key features
  • 4×4 pad grid: assign a sample per pad, trigger with a click, play via the virtual keyboard or any MIDI controller.
  • Instant waveform: quick peak display with Start/End and Loop Start/End markers, plus loop mode indicator (none/forward/bidirectional).
  • Full pad editing: Gain (dB), pan, pitch (±24 semitones), pad mode (Sample/Clip/Slicer/Granular), Poly/Mono, root note and playback options.
  • Drag-and-drop everywhere: from file browser to pad to assign a WAV, between pads to move or copy.
  • Smart preview: Auto-play option when selecting WAVs in the browser.
  • Keyboard/MIDI: play the current pad at any note (with transposition), test mappings instantly with MIDI input.
